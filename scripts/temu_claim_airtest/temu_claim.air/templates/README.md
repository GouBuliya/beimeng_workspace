# 模板采集说明

Airtest 脚本依赖模板图片来识别“认领到”“Temu全托管”与“确定”等按钮。

请在浏览器中打开妙手 ERP 的公用采集箱页面，使用 AirtestIDE 截取以下控件：

| 文件名 | 建议内容 | 备注 |
| --- | --- | --- |
| `toolbar_claim_button.png` | 顶部工具栏中的“认领到”按钮（包含下拉箭头） | 建议在正常亮度下截取，避免透明背景过大 |
| `temu_checkbox.png` | 认领弹窗中 Temu 全托管勾选项（包含文字与勾选框） | 需要包含整行文本，确保能唯一区分 |
| `confirm_button.png` | 认领弹窗底部的“确定”按钮 | 如果按钮有不同状态，可截取默认状态 |
| `dialog_close.png` (可选) | 认领弹窗右上角的关闭图标 | 若流程中偶尔需要关闭弹窗，可提供此模板 |

将截取的 PNG 文件放置在本目录即可。如果控件样式发生变化，请重新截取并覆盖原文件。
