# 产品图片目录

## 使用说明

此目录用于存放产品图片，请按照以下规范命名图片文件：

### 命名规范

**方式一：使用标题后缀命名（推荐）**
```
{标题后缀}_{规格序号}.jpg
```

示例：
- `A026_1.jpg` - 卫生间收纳柜 规格一
- `A026_2.jpg` - 卫生间收纳柜 规格二
- `A026_3.jpg` - 卫生间收纳柜 规格三

**方式二：使用产品名称命名**
```
{产品名称}_{规格序号}.jpg
```

示例：
- `卫生间收纳柜_1.jpg`
- `卫生间收纳柜_2.jpg`

### 支持的图片格式

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

### 在 Excel 中引用

在格式化后的 Excel 文件中，`图片路径` 列填写相对路径：

```
products/A026_1.jpg
products/A026_2.jpg
```

### 自动匹配规则

如果 `图片路径` 列为空，脚本会尝试按以下顺序自动查找图片：

1. `{标题后缀}_{规格序号}.{jpg|png|webp}`
2. `{标题后缀}.{jpg|png|webp}` （单规格产品）
3. 使用默认占位图片

## 当前状态

⚠️ **需要手动添加产品图片**

请将产品图片放入此目录，并在 Excel 中填写对应的路径。

如果原 Excel 中有内嵌图片，可以：
1. 手动从 WPS/金山表格中导出图片
2. 使用截图工具保存图片
3. 从产品来源网站下载原图
